    • 5. Berikan salah satu contoh penggunaan Bootstrap lalu jelaskan apa fungsinya serta berikan screenshot hasil running di web-nya!

  • Bootstrap adalah framework front-end yang membantu membuat halaman web responsif dan konsisten di berbagai ukuran layar, seperti ponsel, tablet, dan desktop, dengan sistem grid yang otomatis menyesuaikan tata letak. Framework ini menyediakan komponen siap pakai seperti tombol, kartu, dan form, yang menjaga keseragaman desain di seluruh halaman tanpa perlu menulis CSS khusus untuk setiap elemen. Selain itu, Bootstrap menawarkan komponen interaktif seperti modal, carousel, dan tooltip untuk meningkatkan interaktivitas tanpa perlu menulis banyak kode JavaScript. Dengan adanya utility classes seperti text-center atau bg-light, Bootstrap memudahkan kustomisasi cepat pada elemen-elemen tampilan. Kompatibilitasnya dengan berbagai browser modern juga memastikan tampilan yang konsisten di berbagai platform. Semua fitur ini menjadikan pengembangan web lebih cepat dan efisien, sehingga developer dapat fokus pada konten dan fungsionalitas
